Use Angular CLI

To add new routes, components, directives, pipes, services, ... see docs of Angular CLI: https://github.com/angular/angular-cli

Angular CLI will even help with upgrading to new Angular2 releases in the future. See video of ng-conf 2016 about Angular CLI: https://www.youtube.com/watch?v=wHZe6gGI5RY

Angular Material 2

We are using Angular Material 2 as a UI framework. The framework has single modules - how to install and add them to the project se here: https://github.com/angular/material2/blob/master/GETTING_STARTED.md

Deploy CCDesktop to edu-sharing base project

Within edu-sharing base project open the 'build-common.xml' ANT file. Start the 'deploy-angular-app' task and follow given directions.
